# 2. Project Overview
Personal Carbon Neutral is an interactive art installation that visualizes personal carbon footprint through physical computing. The installation uses sensors and Arduino to create an engaging, educational experience that encourages environmental awareness.

---

# 3. My Role
- Designed and built the physical installation
- Programmed Arduino microcontrollers for sensor interaction
- Developed Processing visualization system
- Integrated sensors for user interaction

---

# 4. Core Contributions

## 4.1 Interactive Sensor System
The installation features multiple sensors that detect user presence and actions, triggering visual and physical responses.

![Installation](/projects/carbon-neutral/images/cover.png)

This system solves the challenge of making environmental data tangible and engaging.

---

## 4.2 Data Visualization
The installation visualizes carbon footprint data through Processing, creating beautiful and informative displays.

- **Real-time Data**: Live sensor data processing
- **Visual Feedback**: Dynamic visualizations based on user interaction
- **Educational Content**: Clear presentation of environmental impact

![Visualization](/projects/carbon-neutral/images/cover.png)

# 5. Technical Breakdown

## 5.1 System Architecture
